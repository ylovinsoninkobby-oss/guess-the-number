■Guess the Number（数字当てゲーム）■
-このプログラムは、Python を用いて作成した「数字当てゲーム」です。
-ユーザーが 1 から指定した範囲の数字を予想し、正解するまで繰り返す仕組みになっています。

01. 機能
-1〜x の範囲でランダムな数字を生成
-入力した数字が「大きい」「小さい」を日本語でフィードバック
-正解するとメッセージを表示
-難易度（範囲）は guess(100) の数字を変更することで調整可能
-上記の「100」の部分の数字を大きくすれば難易度が上がり、小さくすれば難易度が下がる
-computer_guess(x) を使えば、コンピュータがユーザーの数字を当てるモードも利用可能（現在はコメントアウト）

02.実行方法
-Python をインストール
-ターミナルで以下を実行
-python main.py

03.ファイル構成
